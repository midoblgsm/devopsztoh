We want to implement a shopping cart with the following features:

  • A user should be able to instantiate an empty cart
  • A user can add/remove products to/from his cart
  • A user can consult the content of the cart, unique items or total number of items
  • A user can check the total cost of his cart

What are the types that we need to define here?
