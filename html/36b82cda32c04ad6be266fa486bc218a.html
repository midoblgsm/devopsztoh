To get started, you need first to install docker in your machine. Please follow the specific instructions related to your operating system in this link to install docker. If you are using a linux system, and for your convenience, follow these steps to be able to use docker without the need for sudo in each time. If you are stuck at any point, please try to google it, there are plenty of resources in the internet. If your problem persists, please reach out to me and I can try to help.
