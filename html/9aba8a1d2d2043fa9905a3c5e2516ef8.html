By the end of this course, you will be able to create a distributed application, deploy it through a CICD pipeline following the best practices. You will also have some introductory knowledge of using kubernetes and istio.

The course is not meant to make you an expert in any particular subject but rather introduce you to different topics and give practical use cases that you can use as a starting point to build your experties.

The objectives of the course are the following:

  • Learning the basics of Go coding
  • Understanding how to test drive Go code
  • Being able to containerize and deploy a distributed application
  • Understanding and Setup of a basic CICD pipeline
  • Getting familiar with the features of a service mesh
