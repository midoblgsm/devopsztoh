About This Course

DevOps is a set of practices that reduces the gray line between development and operations. It covers different aspects of software development, tests, deployment and management.

This course will give an introduction to developping distributed applications using Go and Test Driven Development. It will also show how to containerize and deploy the application. Setting up a CICD and a Service Mesh will be also introduced.

Requirements

The students are required to have a basic knowledge of writing simple code.
